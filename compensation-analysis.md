# McPherson Lab - Trainee Compensation Analysis

## Overview
This document outlines the compensation structure for trainees employed at the McPherson Lab. The compensation model consists of a guaranteed base package plus revenue sharing.

## Compensation Structure

### Base Compensation Package

The base package ensures each trainee can maximize tax-advantaged savings, cover rent, and have funds for incidentals:

#### Tax-Advantaged Savings (2026 Contribution Limits)

##### Health Savings Account (HSA)
- **Annual Contribution Limit (2026)**: $4,300 (individual) / $8,550 (family)
- **Assumption**: Individual coverage
- **Annual HSA Contribution**: $4,300

##### 403(b) Retirement Plan
- **Annual Contribution Limit (2026)**: $23,500
- **Annual 403(b) Contribution**: $23,500

##### Roth IRA
- **Annual Contribution Limit (2026)**: $7,000
- **Annual Roth IRA Contribution**: $7,000

**Total Savings**: $34,800

#### Living Expenses
- **Monthly Rent**: $2,000
- **Annual Rent**: $24,000

#### Incidentals
- **Annual Incidentals** (food, transportation, etc.): $12,000

### Base Package Summary

| Category | Annual Amount |
|----------|---------------|
| HSA Contribution | $4,300 |
| 403(b) Contribution | $23,500 |
| Roth IRA Contribution | $7,000 |
| Rent Allowance | $24,000 |
| Incidentals | $12,000 |
| **Base Package Total** | **$70,800** |

### Revenue Sharing

**On top of the base package**, trainees receive **50% revenue sharing** on all revenue they generate through:
- Educational activities (CE courses, live workshops)
- Service laboratory (antibiograms, outbreak detection)

This incentivizes productivity and rewards high performers while ensuring all trainees have their basic needs met.

## How It Works

### Calculating Base Salary Equivalent

To provide the base package of $70,800, we need to calculate the required gross salary that covers:
1. Pre-tax deductions (HSA + 403(b) = $27,800)
2. Post-tax deductions (Roth IRA = $7,000)
3. Taxes (Federal + FICA)
4. Net amount for rent and incidentals ($36,000)

The calculator determines that a gross salary of approximately **$84,706** is required to fully fund the base package after all taxes and deductions.

### Revenue Sharing Example

If a trainee generates $200,000 in annual revenue:
- **Lab's Share (50%)**: $100,000
- **Trainee's Share (50%)**: $100,000

**Total Trainee Compensation**:
- Base Package: $70,800 (in benefits and stipends)
- Revenue Share: $100,000 (taxable income)
- **Total Value**: $170,800

## Notes
- The base package ensures all trainees can maximize tax-advantaged savings and cover basic living expenses
- Revenue sharing provides unlimited upside for high performers
- **IMPORTANT**: This calculation assumes single filing status and **no state/local taxes**
  - State income tax rates vary widely (0% to 13%+ depending on location)
  - Local taxes may also apply in some jurisdictions
  - Actual take-home pay will be lower in states with income tax
- Revenue sharing payments are taxable income and subject to all applicable taxes
